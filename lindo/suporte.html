Introdução

Objetivo deste Manual

Este manual tem como objetivo padronizar os procedimentos do setor de suporte técnico da Conexão Link, garantindo atendimento de excelência aos nossos clientes.

Público-alvo

Técnicos de suporte, analistas de atendimento e qualquer colaborador que interaja diretamente com clientes para solução de problemas técnicos.

Métricas de Qualidade

Nossos principais indicadores de desempenho:

Indicador Meta Periodicidade
TMA (Tempo Médio de Atendimento) ≤ 8 minutos Diário
TME (Tempo Médio de Espera) ≤ 2 minutos Diário
NPS (Net Promoter Score) ≥ 80 Mensal
Resolução na Primeira Chamada ≥ 85% Semanal

Protocolos de Atendimento

Fluxo Padrão de Atendimento

  1. Saudação: "Bom dia/tarde/noite, Conexão Link, [seu nome] falando. Como posso ajudar?"
  2. Identificação: Confirmar nome completo do cliente e número de contrato/CPF
  3. Empatia: Demonstrar compreensão pelo problema relatado
  4. Diagnóstico: Questionar para identificar a natureza do problema
  5. Solução: Aplicar procedimentos técnicos adequados
  6. Validação: Confirmar se o problema foi resolvido
  7. Encerramento: Oferecer ajuda adicional e se despedir

Atenção

Nunca interrompa o cliente durante a descrição do problema. Espere ele concluir antes de iniciar o diagnóstico.

Atendimento a Clientes Especiais

Clientes PCD

  • Oferecer atendimento prioritário
  • Adaptar a linguagem conforme necessidade (ex: falar mais pausadamente para deficientes auditivos)
  • Disponibilizar canal alternativo (WhatsApp ou e-mail) quando necessário

Clientes Idosos

  • Utilizar linguagem simples, sem jargões técnicos
  • Repetir informações importantes
  • Oferecer para enviar instruções por e-mail ou SMS

Procedimentos Técnicos

Problemas Comuns de Conexão

Sem conexão (LED vermelho no modem)

  1. Verificar se há queda na região (consultar sistema OTRS)
  2. Pedir para cliente verificar cabos de alimentação e rede
  3. Solicitar reinicialização do equipamento (desligar por 30 segundos)
  4. Verificar se o MAC está corretamente registrado no sistema
  5. Caso persista, escalar para IXC com ordem de serviço detalhada

Conexão lenta/intermitente

  1. Pedir para cliente testar via cabo (eliminar problema Wi-Fi)
  2. Solicitar teste no speedtest
  3. Verificar potência da fibra no IXC
  4. Analisar histórico de consumo do cliente
  5. Orientar sobre interferências Wi-Fi (canal congestionado, dispositivos próximos)

Trilha do Suporte

AUTORIZAR ONT/ONU:

  1. PROVEDOR > AUTORIZAR ONUs > CONSULTAR TODAS
  2. LOCALIZAR O EQUIPAMENTO INFORMADO PELO TÉCNICO NA COLUNA 'MAC/SERIAL'
  3. SELECIONAR E CLICAR EM 'AUTORIZAR ONU'
  4. PREENCHER AS INFORMAÇÕES DO CLIENTE NOS CAMPOS: CONTRATO E LOGIN
  5. CLICAR EM SALVAR
  6. CASO DÊ ERRO 'LOGIN JÁ POSSUI PERFIL FIBRA' VEJA MAIS ABAIXO, SE NÃO:
  7. INTEGRAÇÃO OLT > GRAVAR NO DISPOSITIVO
  8. AGUARDAR A TELA DE CONFIRMAÇÃO (RETORNO VERDE = SUCESSO)

ERRO: LOGIN JÁ POSSUI PERFIL FIBRA

  1. IR NO CADASTRO DO CLIENTE > LOGIN > FIBRA
  2. SELECIONAR A ONT CADASTRADA > EDITAR
  3. INTEGRAÇÃO OLT > EXCLUIR DISPOSITIVO
  4. AGUARDAR TELA DE CONFIRMAÇÃO

Situações Possíveis:

RETORNO VERDE: Deu tudo certo!
RETORNO VERMELHO:
  1. DELETAR > REFAÇA O PROCESSO DE AUTORIZAÇÃO NOVAMENTE
  2. SE PERSISTIR: PEGAR INFORMAÇÕES EXIBIDAS (SLOT: X PON: X ONU: X)
  3. ALTERAR NOS CAMPOS PON ID: 0/X/X (SLOT/PON)
  4. CORRIGIR 'ONU NUMERO' SE NECESSÁRIO
  5. IR EM INFORMAÇÕES > EDITAR SLOT E PON N > SALVAR
  6. INTEGRAÇÃO OLT > EXCLUIR DISPOSITIVO

    7. Guia Rápido:

    • LIMPAR MAC: CADASTRO > LOGIN > LIMPAR MAC (em toda troca de equipamento)
    • CLIENTE NÃO APARECE NO FIBRA: PROVEDOR > CLIENTES FIBRA > PREENCHER LOGIN
    • ATUALIZAÇÃO FIRMWARE: SALVAR PPPoe > SALVAR WIFI > ADMIN > ATUALIZAR FIRMWARE (1200/121)
    • REAGENDAMENTO: OS > AÇÕES > MARCAR REAGENDAMENTO

    Referência de MACs:

    • 00:19 - CIANET
    • F1 ou 5F no meio - INTELBRAS

    Links Úteis:

    • MAC SEARCH (consultar fabricante pelo MAC)
    • TESTE DE CONEXÃO INTERNO (Senha: 50MD80VP)
    NÃO COMPARTILHAR ESTE LINK COM CLIENTES

    Alteração de Senha Wi-Fi

    Passo a passo:

    1. Acessar interface do roteador (ver tabela acima)
    2. Navegar até Interfaces WLAN > Wifi > Editar Rede Wifi
    3. Selecionar WPA2 como método de segurança
    4. Definir senha com pelo menos 8 caracteres (letras, números e símbolos)
    5. Evitar sequências óbvias ou dados pessoais
    6. Salvar configurações e reiniciar o equipamento

    Dica

    Sugira ao cliente senhas fáceis de lembrar mas difíceis de adivinhar, como "MinhaCasa@2023" ou "FlamengoCampeao#22".
