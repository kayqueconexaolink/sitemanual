Lei Geral das Telecomunicações

Lei nº 9.472/1997 (LGT)

A Lei Geral de Telecomunicações estabelece as bases para organização dos serviços de telecomunicações no Brasil:

Princípios Fundamentais (Art. 3º)

  • Universalização do acesso
  • Continuidade e qualidade dos serviços
  • Modicidade das tarifas
  • Liberdade de escolha do usuário

Direitos dos Usuários (Art. 94)

  1. Qualidade adequada dos serviços
  2. Informações claras sobre tarifas e serviços
  3. Atendimento eficiente às reclamações
  4. Sigilo das comunicações

Obrigações das Empresas

Artigo Obrigação
Art. 61 Autorização da ANATEL para operar
Art. 95 Manter cadastro atualizado de usuários
Art. 171 Não praticar atos lesivos aos usuários

Aplicação Prática no Dia a Dia

Prazos de Reparo (Res. ANATEL 632/2014)

  • Urgentes: 24 horas (ex: queda total do serviço)
  • Normais: 72 horas (ex: instabilidade intermitente)

Portabilidade (Res. ANATEL 718/2019)

  1. Cliente pode migrar entre operadoras
  2. Prazo máximo de 1 dia útil para conclusão
  3. Sem custo adicional para o cliente

LGPD na Telecom

Lei nº 13.709/2018

A Lei Geral de Proteção de Dados estabelece regras para tratamento de dados pessoais:

Dados Sensíveis em Telecom

  • Histórico de chamadas e acessos
  • Localização geográfica
  • Dados de pagamento e consumo
  • Endereço IP e hábitos de navegação

Princípios Fundamentais

Princípio Aplicação
Finalidade Dados só para o propósito informado
Necessidade Coletar apenas o essencial
Livre Acesso Cliente pode consultar seus dados
Segurança Proteção contra acessos não autorizados

Boas Práticas para Colaboradores

O Que NÃO Fazer

  1. Compartilhar dados por WhatsApp ou e-mail não seguro
  2. Manter dados de clientes em planilhas locais
  3. Fornecer informações sem verificar a identidade

Processos Seguros

  1. Verificação de Identidade: Sempre confirmar dados antes de liberar informações
  2. Compartilhamento: Usar apenas sistemas corporativos autorizados
  3. Armazenamento: Manter dados apenas nos sistemas oficiais
  4. Exclusão: Atender solicitações de apagamento conforme política interna
