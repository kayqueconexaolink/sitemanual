Introdução

Objetivo deste Manual

Este manual fornece diretrizes para atendimento SAC, procedimentos comerciais e financeiros da Conexão Link.

Público-alvo

Atendentes SAC, analistas comerciais e financeiros que interagem diretamente com clientes.

Indicador Meta Periodicidade
TMA (Tempo Médio de Atendimento) ≤ 5 minutos Diário
Resolução na Primeira Chamada ≥ 90% Semanal
Satisfação do Cliente (NPS) ≥ 85 Mensal

Cancelamento de Contrato

Fluxo de Cancelamento

  1. Acesso ao Contrato: Localizar o contrato do cliente no sistema IXC
  2. Iniciar Cancelamento: Clicar em Cancel/Negat/Desist
  3. Copiar SERIAL MAC do cliente e colocar na aba "OBSERVAÇÕES" no cadastro.
  4. Efetuar desautorização: Provedor -> Clientes Fibra (ONU) -> Encontrar Cliente -> Desautorizar ONU -> Deletar
  5. Motivo: Selecionar o motivo do cancelamento conforme cenário:
    • Desistiu (pré-acordo)
    • Cancelar contrato (sem faturas futuras)
    • Cancelar e gerar financeiro remanescente (faturas vencidas)
    • Cancelar e gerar financeiro não vencido (proporcional)
  6. Descrição: Inserir resumo da solicitação e informações relevantes
  7. Salvar: Confirmar a operação no sistema

Cálculo Proporcional

Para faturas não vencidas, o sistema calcula os dias utilizados desde o último vencimento até a data do cancelamento, gerando um boleto avulso com o valor proporcional.

Recolhimento de Equipamento

  1. O sistema gera automaticamente uma OS de Recolhimento
  2. Agendar com cliente:
    • Devolução na recepção em 7 dias OU
    • Retirada por técnico (agendar no sistema)
  3. Inserir mensagem: "Agendado recolhimento para dd/mm período (manhã/tarde)"
  4. Selecionar Colaborador Responsável: A definir

2ª Via de Boleto

Envio por E-mail/SMS

  1. Acessar Aba Financeiro no IXC
  2. Selecionar a fatura desejada
  3. Clicar em Imprimir Boletos
  4. Selecionar SIM em "Excluir PDFs antigos"
  5. Escolher método de envio:
    • E-mail: Preencher e confirmar
    • SMS: Selecionar opção correspondente

Confirmação

O sistema exibirá "Finalizou envio com sucesso!" após a operação.

Via WhatsApp (Malu)

  1. Cliente interage com Malu:
    • Opção 4 - Financeiro
    • Opção 1 - Segunda via de boleto
  2. Para múltiplas faturas: Malu oferece opção de escolha

Via OPA

  1. Clicar em Novo Atendimento
  2. Selecionar departamento Financeiro
  3. Digitar nome do cliente
  4. Após interação, selecionar /segundaViaCobrança

Mudança de Titularidade

Requisitos

  • Contrato ATIVO e ADIMPLENTE
  • Fatura vencida: Cliente deve pagar e enviar comprovante
  • Taxa: R$25

Documentação Necessária

Novo titular:
- Selfie + documento de identificação
Antigo titular:
- Documento de identificação

Envio de Documentos

Canal Instruções
E-mail contato@conexaolinkes.com.br
Assunto: "MUDANÇA DE TITULARIDADE + NOME"
WhatsApp (27) 4042-4240
Opção 2 - MUDANÇA DE ENDEREÇO

Processamento

  1. Criar novo cadastro para o novo titular
  2. Gerar boleto avulso da 1ª mensalidade
  3. Registrar em ATENDIMENTO:
    Realizado alteração de titularidade.
    Antigo titular: [Nome + ID]
  4. Agendar técnico para nova configuração do roteador
