O que é a LGPD?

A Lei Geral de Proteção de Dados (Lei 13.709/2018) é a legislação brasileira que regula o tratamento de dados pessoais. Para empresas de telecomunicações, essa lei é especialmente relevante porque:

  • Dados de tráfego e localização são considerados dados pessoais sensíveis
  • Exige transparência no uso dos dados dos clientes
  • Estabelece multas de até 2% do faturamento da empresa (limitado a R$ 50 milhões por infração)
Por que é importante? Na Conexão Link, lidamos diariamente com dados de localização, histórico de navegação e informações de pagamento dos clientes - todos protegidos pela LGPD.

10 Princípios da LGPD

1
Finalidade: Dados devem ser coletados para propósitos específicos e legítimos
2
Adequação: O tratamento deve ser compatível com a finalidade declarada
3
Necessidade: Coletar apenas dados estritamente necessários
4
Livre Acesso: Clientes podem consultar seus dados gratuitamente
5
Qualidade dos Dados: Informações devem ser precisas e atualizadas
6
Transparência: Clientes devem ser informados sobre como seus dados são usados
7
Segurança: Implementação de medidas técnicas e administrativas de proteção
8
Prevenção: Adotar medidas para prevenir danos
9
Não Discriminação: Dados não podem ser usados para fins discriminatórios
10
Responsabilização: A empresa deve comprovar a conformidade

Boas Práticas na Conexão Link

Como aplicar a LGPD no dia a dia:

  • Verificar identidade antes de liberar qualquer informação
  • Não compartilhar dados por telefone sem autenticação
  • Relatar imediatamente qualquer suspeita de vazamento
  • Usar apenas sistemas autorizados para consulta de dados
  • Não anotar dados sensíveis em papéis ou arquivos não seguros

Fluxo para Solicitações de Clientes:

1
Identificação: Verificar documentos que comprovem a identidade
2
Registro: Anotar no sistema a solicitação (direito de acesso, retificação, etc.)
3
Prazo: Resposta deve ser dada em até 15 dias

Dúvidas Frequentes

Posso enviar dados por WhatsApp?

Não. O WhatsApp não é um canal seguro para envio de dados pessoais. Use sempre os sistemas internos autorizados.

E se o cliente pedir para excluir seus dados?

Encaminhe imediatamente para o setor jurídico. Alguns dados precisam ser mantidos por exigência legal (ex: faturas dos últimos 5 anos).
